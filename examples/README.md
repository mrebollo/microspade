# examples — microspade

These are ready-to-use examples for micro:bit.
Flash any of them with the [micro:bit Python editor](https://python.microbit.org/)
or with the [uflash](https://uflash.readthedocs.io/) command-line tool.

| File | Description |
|------|-------------|
| `simple_agent.py` | One agent that scrolls a greeting on the display |
| `ping_pong.py` | Two boards exchanging ping/pong messages |
| `fsm_agent.py` | FSM behaviour that cycles through LED patterns |
